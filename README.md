# react.native.todolist

## Version
- React Native : 16버전 이상
- Redux : 4버전 이상
    
## Project setup
```
yarn install
```

### Run your app
```
yarn run start
```

## Structure
- React Native : 16버전 이상
- Redux : 4버전 이상
- Pages 컴포넌트들의 묶음, 즉 Scene 역할을 한다
    - Init : 최초 페이지
- Components : 최소 컴포넌트 정의
    - FooterComponents : 하단 컴포넌트 (네비게이터) 정의
    - HeaderComponents : 상단 컴포넌트 정의
    - ListComponents : 메인 List형태 컴포넌트 정의
